## [2.0.8] - 2026-05-21

### Removed

- Promotional discount banner ("OSS users get 10% off with code JS26") shown at the top of the editor.
- Mermaid Chart sign-up/login prompts and upsell buttons — "Save diagram", "AI Repair", the "Edit with AI/voice/visually" button, the Playground/Plugins/Mermaid main-menu items, the Share tab's Mermaid Chart option, and the "choose your editor" migration modal (all disabled via the `MERMAID_IS_ENABLED_MERMAID_CHART_LINKS` flag).
- AI prompt popup ("Sign Up at Mermaid.ai to try AI") and its gutter icon in the code editor.
